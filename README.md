## How to use
This code is NYUDv2 implement.

### Requestments
```
Python 3
Pytorch 0.4.1
TensorboardX
Tensorboard
```

### Train the mode

```
python ACNet_train_V1_nyuv2.py --cuda -b 4
```

### Evaluate the mode
The pretrained model for NYUDv2 is avaliable at http://wangkaiwei.org/file/NYUDv2.zip .
